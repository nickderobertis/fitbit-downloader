#  fitbit-downloader

## Overview

A CLI and Python tool for downloading Fitbit data

## Getting Started

Install `fitbit-downloader`:

```
pip install fitbit-downloader
```

A simple example:

```python
import fitbit_downloader

# Do something with fitbit_downloader
```
